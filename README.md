# git_aliases
Create aliases for git commands:

```
co ~> checkout
cb ~> "checkout -b"
ca ~> "checkout -a"
br ~> branch
ba ~> "branch -a",
ci ~> commit
st ~> "status -s"
ft ~> fetch
pl ~> pull
ps ~> push
ad ~> "add ."
cm ~> "commit -m"
in ~> init
lg ~> "log --pretty='> %Cred%h %Creset| %Cgreen"%s"%Creset | %Cblue%an%Creset - %ar' -10"
rv ~> "remote -v"
df ~> diff
```

Access "[output](/output)" folder for windows, mac and linux executables.

or use npm

`npm i -g git_aliases`
